AetherlangLib: Betterphantoms

Betterphantoms provides more tools for working with basic phantoms from base.


phantomrestore ph [indx] @-> array or phantomarray
@ where ph: phantomarray
@       indx: number
@ restores value at INDX if given. returns restored original array if only PH is given.
