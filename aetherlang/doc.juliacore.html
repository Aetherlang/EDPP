AetherlangLib: JuliaCore

An implementation-specific library to interact with the Julia Programming Language. This library is also impossible to use without prefix (typically jl.) since it defines an alternative to typeof from base.


eval exp @-> any
@ where exp: symbol or string
@ evaluates EXP directly in Julia, wraps the result into AetherlangObject (a regular container for values in Aetherlang).

typeof arg @-> string
@ where arg: any
@ returns a string representing the type of ARG in Julia.
